# DataFlowStudio

## Описание на проекта

DataFlow Studio е приложение за обработка и анализ на данни, изградено на принципа на многослойна архитектура. Проектът има за цел да улесни работата с данни чрез структуриран подход, който разделя процеса на няколко логически слоя, всеки със специфична роля.

Приложението използва четири основни слоя – **Source**, **Staging**, **Curated** и **Marts**.  
В слоя Source данните се зареждат автоматично от тестова база данни.  
В слоя Staging се извършва почистване и трансформация на данните чрез заявки, въведени от потребителя.  
Слоят Curated позволява логическо обединяване и структуриране на данните.  
Слоят Marts е предназначен за анализ и визуализация на резултатите.

Проектът предоставя възможности за автоматично генериране на документация, изпълнение на тестове за качество на данните, визуализация на резултати и сравнение на производителността между заявки върху сурови данни и заявки, изпълнени чрез DataFlow Studio.

## Стартиране на проекта

### Изисквания
- Инсталиран **Node.js**  
  https://nodejs.org/en/download
- Подходяща среда за разработка (напр. **Visual Studio Code**)

### Стъпки за стартиране

1. **Отворете проекта**

Отворете проекта в подходяща среда за разработка, например **Visual Studio Code**.

2. **Стартиране на backend частта**

Уверете се, че се намирате в главната папка на проекта(DATAFLOWSTUDIO). Отворете терминал и изпълнете:

cd backend

npm install

npm start

2. **Стартиране на frontend частта**

Уверете се, че се намирате в главната папка на проекта(DATAFLOWSTUDIO). Отворете терминал и изпълнете:

cd giiho-frontend

npm install

npm start
